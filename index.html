Material Design Pull & Sliding Menu with jQuery

Pull and sliding navigation system with Client-Side Functionalities using Material Design, HTML5, CSS3 and jQuery.

Main Features

  • Fully Responsive, auto adoptive height, swipe and pulling support.
  • Chrome, Safari, Firefox, Opera, IE10+, IOS, Android and windows phone supported.
  • Material Design Iconic Fonts.
  • Radial out, fading (in & out), sliding, pushing and pulling effects.
  • Unlimited Main items, dropdowns, sub items can be added.
  • Grid Layouts for Widgets items.
  • Client-Side Notification System. (close able & auto notification counter bage.)

Experimental version

It's an experimental version of "Pull & Sliding Navigation" as I used HTML5 input range element to minimize the complexities and size of code. However, swipe works quite fine (on touch devices and desktop) while opening and closing menu with this logic.
